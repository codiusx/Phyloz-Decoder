## Project Overview
The Phyloz-Decoder framework treats the Voynich Manuscript not as a natural spoken language or a standard substitution cipher, but as a highly optimized, domain-specific compiled database ledger—a medieval *Dispensa* (apothecary registry). The text maps line-by-line to canonical historical compounding sequences.

The theoretical framework, multi-engine compiler stack specification, and full validation metrics are documented in the published manuscript.
